Nylon Demo

This page is served by Nylon's static service.

Useful endpoints

  • http://app.localhost:8088/ → frontend (app-service)
  • http://api.localhost:8088/v1/health → API health (api-service)
  • http://admin.localhost:8088/ → admin panel (admin-service)
  • http://localhost:8088/static/ → this static demo
  • ws://localhost:8088/ws → WebSocket via plugin
  • http://localhost:8088/stream → streaming via plugin
